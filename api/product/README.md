# product Directory

## Overview

- product-model.js: Queries for product-router.js endpoints.
- product-router.js: Endpoints for product-router.

## Endpoints
| **Request Type**| **Product Endpoint** | **Description**|
|-----------------|-------------------|----------------|
| GET | `/api/product/products` | Retrieves all available products. |

## Queries
|**Product Model**|**Description**|
|--------------|---------------------------------------------------|
| getSpecificProduct(product_id) | Retrieves product using product id param |
| getProducts() | Retrieves all available products |
